Contact Links

Whenever you use an email or phone number within your web page, consider using these options to improve your user's experience.

Basic Email Link

This link will automatically open the user's default email program with the TO field already filled out with the proper email address.

<a href="mailto:hello@yoursite.com">Email Me</a>

Email Me:

Basic Email Link with Subject

Auto populate the Subject field in the email program. Spaces in the subject will typically work, but to be sure, you can replace spaces with "%20".

<a href="mailto:hello@yoursite.com?subject=Email%20Subject%20Here">Email Me</a>

Email Me

Adding CC and BCC

Adding CC (Carbon Copy) and BCC (Blind Carbon Copy) can also be added as parameters.

<a href="mailto:hello@yoursite.com?cc=someoneelse@theirsite.com, another@thatsite.com, me@mysite.com&bcc=onelastperson@theirsite.com
&subject=Big%20News">Email Me</a>

Email Me

Styling Email Links

If you wish to style the email links differently than the rest of your page links, a unique pseudo class is available to target links that start with "mailto:".

//  CSS STYLES 
 //  Normal Link 
  a {
    font-family: arial;
    text-decoration: none;
    color: blue;
  }

 //  Email Link 
  a[href^="mailto:"]{
      color: red;
  }

Telephone Links

To force the telephone link to make the call to the number when clicked, which is great for mobile devices, use this code.

// HTML CODE 
  <p><a href="tel:555-555-5555">
555-555-5555</a></p>

// CSS CODE  
  a[href^="tel:"]{
    color: pink;
  }

555-555-5555

Note: The iPhone Safari browser will automatically detect a phone number on a page and will convert the text into a call link without needing this code, but it is a good practice as this will work on: iPhone, Droid / Android, & Blackberry.

Open File Links

Certain web files will open within the browser window when you link to them. These include web files (.html & .css) and images (jpg, .gif, and .png).

    <p><a href="another-page.html">Open another 
    HTML page in Browser</a></p>
    <p><a href="images/chess.jpg">Open Image in
    Browser</a></p>
    <p><a href="style.css">Open CSS in
    Browser</a></p>

Open another HTML page in Browser

Open Image in Browser

Open CSS in Browser

Note: When linking to HTML pages, they will display as you would normally expect. Images (jpg, gif, & png) are displayed at full height and width of the original image. CSS files will simply display the code as it is written and is not usually linked like this.

Download File Links

When non-web files (.zip, pdf, .doc) are linked, those will be downloaded to the userâ€™s computer. If you want the link to download web files, then the attribute download can be added to the link to specify that the file should be downloaded.

Since all files need to follow specific filename rules which can appear messy with the hyphens, etc, the downloaded file can be auto-renamed to use a more friendly filename upon download.

<p>Download with Original Filename: 
<a href="pdf-files/pdf-sample-243.pdf"
 download>Download: Sample PDF File</a></p>

<p>Download with New Filename:
<a href="pdf-files/pdf-sample-243.pdf"
download="SamplePDF File.pdf">
Download: Sample PDF File</a></p>
